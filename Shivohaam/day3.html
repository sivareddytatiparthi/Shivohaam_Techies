CSS Box Model

Margin

Margin is the space outside the border of an element. It creates space between the element and its neighboring elements.

Padding

Padding is the space inside the border of an element. It creates space between the content of the element and its border.

Selector in CSS

  1. ID Selector # => ID selector
  2. Class Selector . => Class selector
  3. Element Selector element => Element selector
  4. Universal Selector * => Universal selector
  5. Grouping Selector element, element => Grouping selector
  6. Descendant Selector element element => Descendant selector
  7. Child Selector element>element => Child selector
  8. Adjacent Sibling Selector element+element => Adjacent sibling selector
